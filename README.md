# [hack-or-snooze](https://nbry.github.io/hack-or-snooze/)

Hack or snooze is a shallow clone of Hacker News. 

API: https://hackorsnoozev3.docs.apiary.io/# 

Current Features:
1. Account Creation
2. Story submission/deletion
3. Favoriting
